# Luxe Haven - Premium Luxury Hotel Website

A stunning, modern luxury hotel website built with Next.js, Tailwind CSS, and Framer Motion. Features a premium dark + gold theme with glassmorphism effects, smooth animations, and full mobile responsiveness.

## ✨ Features

### Design & UI
- **Luxury Theme**: Dark + gold color scheme with elegant typography
- **Glassmorphism**: Modern glass effects with soft shadows
- **Smooth Animations**: Framer Motion powered micro-interactions
- **Responsive Design**: Mobile-first approach with seamless adaptability
- **Cinematic Feel**: 5-star resort aesthetic with parallax effects

### Pages & Sections
1. **Home Page**
   - Fullscreen hero with parallax effects
   - Premium service cards with hover animations
   - Horizontal scrolling gallery
   - Auto-sliding testimonials
   - Video tour section

2. **About Us**
   - Hero section with fade-in animations
   - Split layout storytelling
   - Owner's message with signature styling
   - Mission & vision cards

3. **Gallery**
   - Spiral 3D rotating gallery layout
   - Hover zoom effects with overlay text
   - Fullscreen modal preview
   - Lazy loading optimization

4. **Contact**
   - Luxury contact form with smooth animations
   - Contact information cards
   - Interactive map placeholder
   - Input focus animations

5. **Booking**
   - Premium booking form UI
   - Room type selection with images
   - Real-time price calculation
   - Floating summary card

### Technical Features
- **Next.js 14**: Latest App Router with TypeScript
- **Tailwind CSS**: Utility-first styling with custom luxury theme
- **Framer Motion**: Advanced animations and transitions
- **Responsive**: Mobile-optimized with smooth performance
- **SEO Ready**: Proper meta tags and structure
- **Performance**: Optimized images and lazy loading

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd luxury-hotel-website
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎨 Customization

### Colors
The luxury color palette is defined in `tailwind.config.js`:
- **Gold**: `#d4af37` (luxury-gold)
- **Dark**: `#0a0a0a` (luxury-dark)
- **Charcoal**: `#1a1a1a` (luxury-charcoal)

### Typography
- **Headings**: Playfair Display (serif)
- **Body**: Inter (sans-serif)

### Animations
Custom animations are defined in the Tailwind config:
- `fade-in`: Smooth fade with slide up
- `slide-up`: Upward slide animation
- `float`: Gentle floating effect
- `glow`: Luxury glow effect
